Conceptos HTML y CSS

Propiedades de texto

Las propiedades de texto en CSS sirven para controlar cómo se ve el texto en una página web. Con ellas puedes cambiar aspectos como el color, el tamaño de la letra, el tipo de fuente, el espaciado y la alineación del texto. Por ejemplo, puedes hacer que un párrafo esté centrado, que las letras sean más grandes o que el texto tenga un color específico. Estas propiedades ayudan a que el contenido sea más legible, organizado y visualmente atractivo para las personas que visitan el sitio web.

FlexBox

Las propiedades de CSS Flexible Box Layout sirven para controlar cómo se organizan los elementos dentro de un contenedor en una página web. Con ellas se puede cambiar la dirección, la alineación y el espacio entre los elementos. Por ejemplo, se puede hacer que los elementos estén en fila, en columna o centrados dentro del contenedor. Estas propiedades ayudan a que el diseño de una página sea más ordenado, flexible y fácil de adaptar a diferentes pantallas.

Medidas

En CSS, rem es una unidad de medida basada en el tamaño de letra del elemento html. Los navegadores suelen usar 16px por defecto, pero al establecer html { font-size: 62.5%; }, 1rem equivale a 10px. Esto se utiliza porque facilita los cálculos de tamaños y ayuda a mantener el diseño más ordenado y adaptable.

Box Model

El Box Model en HTML y CSS describe cómo se estructura el espacio de un elemento. Cada elemento se compone de contenido, padding (espacio interno), border (borde) y margin (espacio externo). Este modelo permite controlar el tamaño y la separación de los elementos dentro de una página web.

Display

La propiedad display en CSS define cómo se muestra un elemento en la página. Permite controlar si un elemento se comporta como bloque, línea o contenedor flexible. Por ejemplo, block ocupa todo el ancho disponible, inline solo el espacio necesario y flex facilita organizar elementos dentro de un contenedor.
